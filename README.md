# 🚗 Campus Carpooling System

## 📌 Overview
The Campus Carpooling System is a web-based application developed during a hackathon to help students share rides efficiently. The platform reduces transportation costs, minimizes traffic congestion, and promotes eco-friendly travel within campus.

---

## 🌐 Live Demo
🔗 https://tejasvidiwan03.github.io/campus-carpool/

---

## 🎯 Problem Statement
Many students travel individually to campus, leading to higher expenses and increased traffic. There is a need for a system that connects students traveling in the same direction.

---

## 💡 Solution
This project provides a platform where students can:
- Find available rides
- Share rides with others
- Reduce travel costs
- Travel in a more eco-friendly way

---

## ⚙️ Features
- Ride matching system
- Simple and user-friendly interface
- Cost-sharing concept
- Efficient route coordination
